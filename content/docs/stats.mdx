---
title: Workspace Stats
description: Retrieve top-level workspace counts from your Vine workspace
---

The Workspace Stats API returns summary counts for the workspace connected to your API key.

## Fetch Workspace Stats

Retrieve summary totals for public posts, authors, categories, and tags.

### Link

```text
https://vinecms.tech/api/public/v1/{API_KEY}/stats
```

### Endpoint

```text
GET /api/public/v1/{API_KEY}/stats
```

### Query Parameters

None right now.

### Example Request

```bash
curl "https://vinecms.tech/api/public/v1/{API_KEY}/stats"
```

### Example Response

```json
{
  "ok": true,
  "workspace": {
    "id": "workspace_id",
    "name": "Vyeos",
    "slug": "vyeos"
  },
  "stats": {
    "totalPosts": 12,
    "totalAuthors": 3,
    "totalCategories": 5,
    "totalTags": 9
  }
}
```

### Notes

- `totalPosts` currently counts only posts that are both `published` and `visible`
- the other totals count all authors, categories, and tags in the workspace

### Use Cases

- public workspace summary cards
- marketing site counters
- external dashboards

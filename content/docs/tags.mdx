---
title: Tags
description: Retrieve tags from your Vine workspace
---

The Tags API returns the tags available in the workspace connected to your API key.

## Browse All Tags

Retrieve the full list of tags for a workspace.

### Link

```text
https://vinecms.tech/api/public/v1/{API_KEY}/tags
```

### Endpoint

```text
GET /api/public/v1/{API_KEY}/tags
```

### Query Parameters

None right now.

### Example Request

```bash
curl "https://vinecms.tech/api/public/v1/{API_KEY}/tags"
```

### Example Response

```json
{
  "ok": true,
  "workspace": {
    "id": "workspace_id",
    "name": "Vyeos",
    "slug": "vyeos"
  },
  "tags": [
    {
      "id": "tag_id",
      "name": "Updates",
      "slug": "updates",
      "createdAt": "2026-03-08T10:00:00.000Z"
    }
  ]
}
```

### Use Cases

- tag cloud pages
- search facets
- editorial taxonomy tools

---
title: Categories
description: Retrieve categories from your Vine workspace
---

The Categories API returns the categories available in the workspace connected to your API key.

## Browse All Categories

Retrieve the full list of categories for a workspace.

### Link

```text
https://vinecms.tech/api/public/v1/{API_KEY}/categories
```

### Endpoint

```text
GET /api/public/v1/{API_KEY}/categories
```

### Query Parameters

None right now.

### Example Request

```bash
curl "https://vinecms.tech/api/public/v1/{API_KEY}/categories"
```

### Example Response

```json
{
  "ok": true,
  "workspace": {
    "id": "workspace_id",
    "name": "Vyeos",
    "slug": "vyeos"
  },
  "categories": [
    {
      "id": "category_id",
      "name": "Product",
      "slug": "product"
    }
  ]
}
```

### Use Cases

- content taxonomy pages
- category pickers
- navigation and section pages

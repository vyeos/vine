This page provides TypeScript reference types for the currently shipped Vine public API.

## Shared Types

### ErrorResponse

All error responses follow this format:

```ts
type ErrorResponse = {
  ok: false;
  error: string;
};
```

## Posts

### PostSummary

Used in post lists and includes basic post information.

```ts
type PostSummary = {
  id: string;
  title: string;
  slug: string;
  excerpt: string;
  readingTimeMinutes: number;
  publishedAt: string | null;
  updatedAt: string;
  author: { id: string; name: string } | null;
  category: { slug: string; name: string } | null;
  tags: Array<{ slug: string; name: string }>;
};
```

### PostDetail

Used when fetching a single post. Includes all fields from PostSummary plus the full HTML content.

```ts
type PostDetail = PostSummary & { contentHtml: string };
```

### PostListResponse

Response shape for post list endpoints.

```ts
type PostListResponse = {
  ok: true;
  posts: PostSummary[];
};
```

### PostDetailResponse

Response shape for post detail (single post) endpoints.

```ts
type PostDetailResponse = {
  ok: true;
  post: PostDetail;
};
```

## Authors

### Author

```ts
type Author = {
  id: string;
  name: string;
  email: string;
  about: string;
  socialLinks: Record<string, string>;
};
```

### AuthorListResponse

Response shape for authors endpoint.

```ts
type AuthorListResponse = {
  ok: true;
  authors: Author[];
};
```

## Workspace Stats

### Stats

```ts
type Stats = {
  totalPosts: number;
  totalAuthors: number;
  totalCategories: number;
  totalTags: number;
};
```

### StatsResponse

```ts
type StatsResponse = {
  ok: true;
  stats: Stats;
};
```

## Operational Checks

### HealthResponse

```ts
type HealthResponse = {
  ok: true;
  service: 'web';
  runtime: 'nextjs+elysia';
};
```

### StatusResponse

```ts
type StatusResponse = {
  ok: true;
  message: string;
};
```

## Notes

- `readingTimeMinutes` is estimated from text, images, and code blocks
- route-level failures use `error`, not `message`
- the current API does not ship pagination or filtering response metadata

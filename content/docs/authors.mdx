---
title: Authors
description: Retrieve author information from your Vine workspace
---

The Authors API returns the authors available in the workspace connected to your API key.

## Browse All Authors

Retrieve the full list of authors for a workspace.

### Link

```text
https://vinecms.tech/api/public/v1/{API_KEY}/authors
```

### Endpoint

```text
GET /api/public/v1/{API_KEY}/authors
```

### Query Parameters

None right now.

### Example Request

```bash
curl "https://vinecms.tech/api/public/v1/{API_KEY}/authors"
```

### Example Response

```json
{
  "ok": true,
  "workspace": {
    "id": "workspace_id",
    "name": "Vyeos",
    "slug": "vyeos"
  },
  "authors": [
    {
      "id": "author_id",
      "name": "Ava Stone",
      "email": "ava@example.com",
      "about": "Writes product and editorial content.",
      "socialLinks": {
        "x": "https://x.com/ava"
      }
    }
  ]
}
```

### Use Cases

- author directory pages
- author dropdowns in external apps
- profile cards and bylines
